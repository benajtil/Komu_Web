# KOMU – Local Community Social Platform

A barangay-based social media platform that allows users to:
- interact within their local community
- post updates
- verify identities via barangay officials
- search within local scope

## Tech Stack
- Frontend: React (Vite)
- Backend: PHP
- Database: PostgreSQL

## Features
- Barangay-scoped feed
- Role-based access (admin, officials)
- Verification system
- Local search (people, posts, comments)
- Profile system

## Note
Sensitive files and uploads are excluded for security.
